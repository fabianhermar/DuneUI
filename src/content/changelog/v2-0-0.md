After months of work, we're thrilled to ship **DuneUI v2.0** — the most significant update since launch. This release upgrades the entire component library to TailwindCSS v4, introduces Astro as the backbone of the docs site, and brings a fresh visual identity.

## What's new

- Migrated the entire component library to **TailwindCSS v4** syntax
- Introduced **Astro** as the main framework powering the documentation site
- New **Blog section** with full MDX support
- Added **40+ new UI components** across Application UI, Marketing, and Ecommerce categories
- Revamped landing page with animated hero, smooth scroll effects, and new sections

## Improvements

- Significantly improved **accessibility** across all interactive components — proper ARIA labels, focus rings, keyboard navigation
- Better **responsive behavior** on mobile and tablet viewports
- Faster preview rendering for component examples

## Breaking changes

- Removed all legacy CSS utility classes in favor of TailwindCSS v4 syntax. If you've been copying classes from v1.x, you may need to update them.
- The component preview markup now uses semantic HTML5 elements throughout
