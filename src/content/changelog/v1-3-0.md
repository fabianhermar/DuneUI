This release focuses on breadth. We added a dedicated Icons section, shipped the first set of full-page templates, and dropped a bunch of new Ecommerce components that were heavily requested by the community.

## What's new

- **Icons section** — 200+ hand-picked icons, searchable and copy-paste ready
- First set of **full-page templates**: Landing Page, SaaS, and Portfolio starters
- New Ecommerce components: **Shopping Cart**, **Product Detail**, **Reviews & Ratings**, **Order Summary**
- Optimized SVG rendering performance for icon previews

## Bug fixes

- Fixed dropdown z-index issue that caused overlapping on certain page layouts
- Corrected inconsistent padding in form components across different screen sizes
- Fixed copy-to-clipboard not working on Safari 17
