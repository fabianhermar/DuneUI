DuneUI has always been great for marketing pages. With v1.1, we're making it equally powerful for internal tools, dashboards, and SaaS applications.

## What's new

- **Tables** — with sorting indicators, pagination controls, and a polished empty state
- **Form Layouts** — stacked, two-column, and card-based form variants
- **Select Menus** — native and custom styled
- **Radio Groups** and **Checkbox Groups** — with icon and description variants
- **Navbars** — top navigation with responsive mobile drawer
- **Pagination** — with ellipsis support and accessible labels
- Component previews now support a **dark mode toggle**

## Improvements

- Refactored all form components to use consistent spacing and label positioning
- Preview code is now formatted using Prettier on render

## Bug fixes

- Fixed copy-to-clipboard not working correctly on Safari browsers
