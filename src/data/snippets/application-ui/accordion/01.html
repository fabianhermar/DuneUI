What is DuneUI?

DuneUI is a modern, beautifully designed component library built with Tailwind CSS. It allows you to easily copy and paste components into your projects.

Do I need React?

No, DuneUI components are provided as raw HTML and Tailwind CSS, meaning they can be used in any framework or even vanilla HTML projects.

Is it fully responsive?

Yes! Every component is built with mobile-first design principles using Tailwind's responsive utility classes.
